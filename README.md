#📁 Download Organizer / Органайзер загрузок

Автоматическая сортировка хаоса в папке «Загрузки». Этот скрипт раскладывает файлы по папкам: **Изображения**, **Документы**, **Музыка**, **Видео**, **Архивы** и **Программы**.

Больше никаких сотен файлов в одной куче!

✨ Возможности

- 🖼️ Сортировка по типам файлов (более 30 расширений).
- 🗂️ Создание отсутствующих папок автоматически.
- 🚫 Игнорирование системных и уже разложенных файлов.
- 📝 Логирование действий (что куда перемещено).
- ⚙️ Настраиваемые правила сортировки (легко добавить свой тип).
- 🐍 Кроссплатформенность (Windows, macOS, Linux).

📸 Пример работы

**До:**
`Загрузки/` → `photo.jpg`, `report.pdf`, `song.mp3`, `setup.exe`, `archive.zip`

**После:**
Загрузки/
├── Изображения/ → photo.jpg
├── Документы/ → report.pdf
├── Музыка/ → song.mp3
├── Программы/ → setup.exe
└── Архивы/ → archive.zip

🚀 Быстрый старт
Требования
- Python 3.7 или выше
- Операционная система: Windows / Linux / macOS

Использование
1 Запуск один раз:
python organizer.py
2 Автоматический запуск (каждые N минут):
python organizer.py --watch 30   # проверка каждые 30 минут
3 Указать другую папку:
python organizer.py --path "C:\Users\Имя\Downloads"

⚙️ Конфигурация
Отредактируйте файл config.json, чтобы изменить правила сортировки:
{
  "downloads_folder": "~/Downloads",
  "categories": {
    "Images": [".jpg", ".jpeg", ".png", ".gif", ".bmp", ".svg"],
    "Documents": [".pdf", ".docx", ".txt", ".xlsx", ".pptx", ".md"],
    "Audio": [".mp3", ".wav", ".flac", ".aac"],
    "Video": [".mp4", ".mkv", ".mov", ".avi"],
    "Archives": [".zip", ".rar", ".7z", ".tar", ".gz"],
    "Executables": [".exe", ".msi", ".dmg", ".appimage", ".sh"],
    "Code": [".py", ".js", ".html", ".css", ".cpp", ".json"]
  },
  "ignore_extensions": [".tmp", ".crdownload", ".part"]
}

🛠️ Запуск по расписанию (без сторонних библиотек)
Windows (Планировщик заданий)
Создайте задачу в Планировщике заданий с триггером "При входе в систему" или "Ежедневно".

Linux/macOS (Cron)
Добавьте строку в crontab -e:
*/30 * * * * /usr/bin/python3 /путь/к/organizer.py
